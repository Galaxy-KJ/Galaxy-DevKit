TypeScript Example (read-only)

TypeScript example showing how to call an HTTP API from Node or a frontend toolchain. Copy into your project and adapt the endpoint and auth headers.

// Example: fetch a Galaxy DevKit REST endpoint (Node 18+)
import fetch from 'node-fetch';

const endpoint = 'https://api.example.com/v1/resource';

async function main(){
  const res = await fetch(endpoint, { method: 'GET', headers: { 'Accept': 'application/json' } });
  if (!res.ok) throw new Error(`HTTP ${res.status}`);
  const data = await res.json();
  console.log(data);
}

main().catch(err => console.error(err));
