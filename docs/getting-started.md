# Getting Started with Galaxy DevKit

This guide will take you from a clean terminal to submitting your first smart wallet transaction on the Stellar Testnet.

## 1. Prerequisites
Before starting, ensure you have:
* **Node.js**: v18.0.0 or higher.
* **Stellar Account**: A testnet public/private key pair.
* **Funds**: Use [Friendbot](https://laboratory.stellar.org/#friendbot) to fund your account with test XLM.

## 2. Installation
Install the core wallet package:
```bash
npm install @galaxy-devkit/core-wallet
